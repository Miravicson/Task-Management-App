# Miravicson Task Management App 
## Description

* [Heroku Backend Deploy](https://miravicson-task-management.herokuapp.com)
* [Swagger Docs](https://miravicson-task-management.herokuapp.com/docs)
* [GitHub Pages Frontend Deployment](https://miravicson.github.io/task-management-frontend)

## Tech Stack

* NestJs
* PostgreSQL
* Jest
* TypeORM
* Express
* JWT Authentication

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
